# MFFO

Orderflow for Mamacita fastfood.

## Requirements

- Node.js 22+
- npm 11+

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Test

```bash
npm run test
```

## Lint

```bash
npm run lint
```

## Test Environment

When the app starts, choose either `Butcher Burgers` or `Tacos` on the landing page.
The selection controls which menu JSON is used for the same customer, kitchen, and pickup flow.
